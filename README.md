# Portfolio Performance Risk Analysis
## The Mountain Path - World of Finance

A comprehensive risk analysis tool for equity portfolios using advanced quantitative metrics.

### Features
- Value at Risk (VaR) & Expected Shortfall
- Maximum Drawdown Analysis
- Sharpe, Sortino, Calmar, Information Ratios
- Live Nifty 50 stock data
- Interactive portfolio weight allocation
- Multi-tab analysis interface

### Installation
```bash
pip install -r requirements.txt
```

### Usage
```bash
streamlit run app.py
```
